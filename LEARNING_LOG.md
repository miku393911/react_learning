## Part 1: ロジックと条件分岐

### Q1. カウンター
*   **出題内容**: ボタンをクリックすると数字が1ずつ増えるカウンター
*   **使用したHooks**: `useState`
*   **学んだこと・気づき**:
    *   (ここに自身の言葉で記入)
    *   例: stateの更新は非同期で行われること
*   **コードのポイント**:
    ```jsx
    // 例
    const handleClick = () => {
      setCount(prev => prev + 1);
    };
    ```

### Q2. トグルボタン
*   **出題内容**: ボタンクリックでON/OFF切り替え
*   **学んだこと**:
    *   boolean値の反転には `!prev` が使える
*   **コードのポイント**:
    ```jsx
    setIsOn(prev => !prev);
    ```
